## 2026-04-20

## ✅ 今日完成功能
1.  **智慧個股辨識**：
    - 引入 `StockExtractor` 類別，使用正則表達式精準抓取 4 位數台股代號。
    - 建置 `stock_mentions` 關聯表，支援個股統計功能。
2.  **資料庫遷移 (Migration)**：
    - 新增 `has_stock_code` 與 `has_link` 欄位。
    - 執行 Backfill 腳本，自動掃描並標註舊有訊息中的連結與股票。
3.  **多媒體過濾與連結偵測**：
    - 匯入時自動跳過 `[照片]`、`[貼圖]` 等佔位符。
    - 支援 `Links Only` (僅顯連結) 篩選模式。
4.  **聊天室管理**：
    - 新增「重新命名」功能。
    - 新增「移除聊天室」功能（連動刪除訊息與統計資料）。
5.  **Git 版本管理**：
    - 初始化 Git 倉庫，設定 `.gitignore` 以保護 `chat.db` 隱私。
    - 解決 Windows 環境下的 Git 安裝與 PATH 配置問題。

## 🛠️ 技術細節
- **FTS5 搜尋**：維持使用 Trigram 分詞，確保繁體中文搜尋的高性能。
- **資料完整性**：刪除聊天室時採用 Transaction 確保關聯表資料同步清除。
